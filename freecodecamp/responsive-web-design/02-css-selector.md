# CSS Selector
Multiple elements can be selected together with the use of a comma (`,`).
```css
element1, element2 {
    css-property: value;
}
```

Use a hash (`#`) to select a specific element with an `id` attribute.
```html
<div id="menu"></div>
```

```css
#menu {
    width: 300px;
}
```

Use a period (`.`) to select a specific element with a `class` attribute.
```html
<div class="menu"></div>
```

```css
.menu {
    width: 300px;
}
```
