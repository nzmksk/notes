## Go Data Types
### Boolean Data Type
- Declared with `bool` keyword.
- Can only take values `true` or `false`.

### String Data Type
- Declared with `string` keyword.
- Must be surrounded by **double quotes** only.

### Default Values of Data Type
| Data Type | Default Type | Default Value |
| --- | --- | --- |
| `bool` | `bool` | false |
| `int` | `int` | 0 |
| `float` | `float64` | 0 |
| `string` | `string` | "" |
